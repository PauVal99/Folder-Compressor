Detailed Description

Esta clase representa el algoritmo de compresión y descompresión JPEG. Se encarga de comprimir y descomprimir imagenes PPM, haciendo uso de la transformada discreta de coseno y Huffman para una mayor compressión.

Author
Bartomeu Perelló
